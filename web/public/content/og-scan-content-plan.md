# OG Scan — Content Ideas, Structure & Planning
## For Content Manager / Social Media Team

---

## CONTENT PILLARS

### 1. 🔧 Product Updates (Feature Drops)
- Announce new features with screenshots/screen recordings
- "What's new this week" roundups
- Before/after comparisons

### 2. 🛡️ Trust & Security
- Highlight anti-fraud, LP detection, device tracking
- "We protect our community" messaging
- Security tips for meme coin traders

### 3. 🎤 Community & Social
- Spaces highlights, voice room recaps
- User spotlights, badge showcases
- Referral leaderboard shoutouts

### 4. 📊 Alpha & Education
- How to use the scanner, what LP pulled means
- Reading token metrics, spotting rugs
- Wallet analysis tips

### 5. 🚀 Growth & Hype
- Invite system promotion, XP rewards
- Milestone celebrations (user count, scans)
- Teaser threads for upcoming features

---

## CONTENT FORMATS THAT WORK

| Format | Platform | Notes |
|--------|----------|-------|
| Feature thread (5-8 tweets) | Twitter/X | One feature deep-dive per thread |
| Screenshot + caption | Twitter/X | Quick visual update |
| Short video (15-30s) | Twitter/X, TikTok | Screen recording of feature in action |
| Comparison graphic | Twitter/X | Before/after or us vs. others |
| Poll | Twitter/X | Engagement bait: "Which feature do you want next?" |
| Meme | Twitter/X | Use OG Scan meme gallery for inspiration |
| Infographic | Twitter/X | Stats, numbers, achievements |

"1/ [Specific capability]

"Try it now → ogscan.fun

## ENGAGEMENT TACTICS
1. Always end with a question or CTA
2. Reply to comments within 1 hour
3. Quote-tweet community members using OG Scan
4. Pin the most important weekly thread
5. Use polls to let community vote on next features
6. Screenshot and share leaderboard milestones
