## IMAGE STYLE GUIDE
- **Dark backgrounds** — match the OG Hacker theme (#0a0a0f, #0f0f15)
- **Accent colors** — Cyan (#22d3ee), Lime (#84cc16), Gold (#eab308)
- **Font vibe** — Monospace for stats/data, bold sans-serif for headlines
- **Screenshots** — Use mobile view (most users are mobile), crop tight, add subtle border glow
- **Graphic style** — Minimal, dark, techy. Think terminal aesthetic meets premium crypto app.
