THE 12 NON-NEGOTIABLE PRINCIPLES
01 First Principles Over Patterns
Deconstruct every UI to its purpose. Ask “Why does this exist?” before choosing any component, color, or animation.
02 Whitespace is Sacred
Generous, intentional negative space. Never crowd. The eye needs room to rest. Apple-level breathing room is the baseline.
03 Typography as Architecture
Tracking, weight, and scale create hierarchy and emotion. Use system fonts with precision. Headlines should feel inevitable.
04 One Primary Action Per View
Clarity archetype: Ruthlessly reduce cognitive load. The user should never wonder what to do next.
05 Information Density Without Chaos
Forge & Lumina: Rich data is welcome when perfectly organized. Use visual weight, alignment, and subtle cards.
06 Micro-Interactions That Soothe
Every hover, transition, and state change must feel deliberate and premium. 200-400ms cubic-bezier timing. Never janky.
07 Dark Elegance Default
Deep charcoal backgrounds (#0A0A0B, #121214). Gold (#C9A959) or refined indigo accents. High contrast text. Luxury tech feel.
08 Interactive by Default
Buttons do things. Inputs respond instantly. Live updates feel alive. Toast systems, loading states, and feedback are mandatory.
09 Hybrid Brand DNA
Synthesize the best of Apple (minimal), Best Buy (trust + commerce), OG SCAN (precision command), NBA (energy & data). Never copy — synthesize.
10 Self-Contained HTML5
Single file. Tailwind via CDN + vanilla JS. Chart.js when needed. No external dependencies unless explicitly required. Perfect for training data.
11 Long-Horizon Craft
Every element must feel like it was considered for 5 years of use. No trends. Timeless quality that ages gracefully.
12 Adversarial Self-Review
Before finishing, ask: “Would this embarrass me if a world-class designer reviewed the source?” If yes, refine until the answer is no.
